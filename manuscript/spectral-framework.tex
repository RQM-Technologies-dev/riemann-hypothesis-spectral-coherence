This section defines a minimal formal interface for the manuscript's proposed operator/coherence layer.
It does not claim that the layer is complete.
Its role is to make explicit which properties must be supplied before downstream RH-reduction claims can be treated as theorem-level statements.

\subsection*{Current formal status and canonical choice}
\placeholder{A finalized operator-level construction has not yet been fixed.}
The manuscript now fixes one canonical predicate for the main text:
\[
\text{Bal}_H(s)\iff \Phi_H(s)=0 \quad (s\in\mathcal{A}),
\]
where $H$ is the proposed spectral object and $\Phi_H$ is the balance functional derived from that object.
At present, $H$ and $\Phi_H$ are formalization targets rather than complete theorem-level objects.

\subsection*{Minimal required structure for the $H$-layer}
The manuscript does not need every possible operator-theoretic feature.
It does need the following \emph{minimal interface} to keep later arguments well-typed and auditable.

\paragraph{(M1) Ambient space and typing data.}
A specified complex topological vector space (typically a Hilbert or Banach model) $\mathcal{X}$, together with clear codomain conventions for all derived maps.
Every use of $H$ must state its typing context explicitly.

\paragraph{(M2) Domain and well-defined action.}
A dense or otherwise explicitly justified domain $\mathcal{D}(H)\subseteq\mathcal{X}$ (as required by the chosen model), and a map
\[
H:\mathcal{D}(H)\to\mathcal{X}
\]
with proof that this action is well defined under stated hypotheses.
If closure/closability is needed by subsequent lemmas, that requirement must be stated and proved where used.

\paragraph{(M3) Symmetry-compatibility package.}
A precise compatibility statement linking the operator layer to the reflection involution $\sigma(s)=1-s$ from \texttt{definitions.tex}.
This must be explicit and non-heuristic.
Informal symmetry pictures are admissible only as motivation, not as substitute for this package.

\paragraph{(M4) Operator-regularity assumptions (structural side).}
The exact regularity required for operator statements: continuity/measurability/analytic dependence of operator-valued maps, domain persistence, and closure/graph conditions needed for later lemmas.
Each assumption must be tagged as either proved in-manuscript or imported as an explicit hypothesis.

\paragraph{(M5) Dependency traceability.}
A lemma-level dependency map from operator properties to later claims (especially Targets~A and~B).
No RH-relevant step should rely on an unlisted implicit property of $H$.

\subsection*{Minimal required structure for the canonical functional \texorpdfstring{$\Phi_H$}{Phi_H}}
Once a candidate $H$-layer is fixed, the manuscript uses one canonical functional $\Phi_H$ with explicit dependence on that layer.

\paragraph{(P1) Definition from finalized inputs.}
A non-circular definition rule for $\Phi_H(s)$ using only previously defined data.
The definition must not presuppose the zero set of $\zeta$.

\paragraph{(P2) Domain of evaluation (admissible class).}
A specified admissible class $\mathcal{A}\subseteq\C$ consistent with \texttt{definitions.tex}, including all side conditions needed for finiteness and well-posedness.
If $\Phi_H$ is only partially defined, the excluded region must be explicit.

\paragraph{(P3) Well-posedness statement.}
For $s\in\mathcal{A}$, proof that $\Phi_H(s)$ exists and is finite, plus the exact continuity/regularity properties later used in correspondence and exclusion steps.
Computational checks may support plausibility but do not satisfy this requirement.

\paragraph{(P4) Symmetry behavior of \texorpdfstring{$\Phi_H$}{Phi_H}.}
A formal statement of how $\Phi_H$ transforms under $s\mapsto1-s$, derived from stated $H$-properties.
This is required for reflection-based arguments.

\paragraph{(P5) Bridge interface to zero claims.}
A precise theorem target describing what must be shown about
\[
\Phi_H(s)=0
\]
and how that condition maps to encoded zeta-zero statements.
This interface should separate:
\begin{enumerate}
  \item the forward direction to be proved,
  \item the converse direction to be proved,
  \item side conditions preventing spurious balanced points.
\end{enumerate}

\subsection*{Canonical exclusion target supported by this interface}
Let
\[
\mathcal{A}_{\mathrm{strip}}:=\{s\in\mathcal{A}:0<\RePart(s)<1\},
\qquad
\mathcal{B}:=\{s\in\mathcal{A}_{\mathrm{strip}}:\Phi_H(s)=0\}.
\]
The main-text exclusion burden is the single subset statement
\[
\mathcal{B}\subseteq\{s\in\mathcal{A}_{\mathrm{strip}}:\RePart(s)=\tfrac12\}.
\]
This section supports that statement by listing operator and analytic prerequisites.
It does not claim those prerequisites are proved.

\subsection*{Open support packages (dependency statements, not proved results)}
\paragraph{Operator-side package (OP).}
The following remain open theorem targets: well-defined operator action and closure properties, symmetry-compatible transport on relevant domains, legitimacy of any invoked spectral constructions, compatibility of operator transforms with evaluation of $\Phi_H$, and dependency-locked usage across Targets~A/B.

\paragraph{Analytic-control package (AC).}
The following remain open theorem targets: strip-wide bound control, uniform convergence where transfer is used, legitimacy of limit/interchange operations, continuation control from local to strip-wide regimes, and exclusion of hidden off-line branches of $\mathcal{B}$ caused by analytic degeneration.

\paragraph{Status note.}
OP and AC are recorded as open bridge classes for audit.
They specify proof burden only.
They do not provide a proof of Target~A or Target~B.

\subsection*{Why this layer is necessary for the RH reduction}
The conditional reduction step in \texttt{main-results.tex} depends on theorem-level statements about $H$ and $\Phi_H$.
Without a rigorous $H$/$\Phi_H$ layer:
\begin{enumerate}
  \item the predicate $\Phi_H(s)=0$ is not a theorem-level condition on a defined domain,
  \item the correspondence target cannot be stated non-circularly,
  \item the canonical exclusion target has no formal object to which estimates apply.
\end{enumerate}
Therefore this layer is a prerequisite interface for the manuscript's proposed RH reduction route.

\begin{heuristic}
Geometric or cancellation intuition can guide candidate constructions of $H$ and $\Phi_H$.
However, such intuition remains heuristic until converted into explicit definitions, hypotheses, and proved lemmas matching the interface above.
\end{heuristic}
