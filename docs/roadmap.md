# Roadmap

*The planned publication and formalization strategy for this research program.*

---

## Phase Overview

The proposed publication sequence consists of the following phases:

### Phase 1 — Clean Public Repository *(current stage)*

- [x] Repository structure established
- [x] Documentation scaffolded (README, claim map, overview, notation, FAQ)
- [x] Issue templates for mathematical review
- [x] GitHub Actions for LaTeX build and link checking
- [ ] Manuscript draft: all sections populated with substantive content
- [ ] Known gaps fully documented in `notes/known-gaps.md`
- [ ] Objections and responses populated in `notes/objections-and-responses.md`

### Phase 2 — Stable Manuscript

- [ ] All manuscript sections contain complete, consistent mathematical content
- [ ] All theorem environments are filled with either proofs or explicit placeholders
- [ ] The spectral operator H is precisely defined
- [ ] The correspondence between zeros of ζ and balance points of H is formally stated
- [ ] The critical-line argument is complete or the gap is clearly marked
- [ ] Manuscript compiles cleanly to PDF via GitHub Actions
- [ ] Internal cross-references are consistent

### Phase 3 — Code and Visual Support

- [ ] Spectral visualization notebook (`code/notebooks/spectral-visualization.ipynb`) complete
- [ ] Critical-line geometry notebook (`code/notebooks/critical-line-geometry.ipynb`) complete
- [ ] Key figures generated programmatically and stored in `manuscript/figures/`
- [ ] Code is documented and reproducible

### Phase 4 — Private Expert Review

- [ ] Manuscript sent to at least two mathematicians with expertise in analytic number
      theory or spectral theory for private review
- [ ] Review comments incorporated or explicitly addressed
- [ ] `notes/review-log.md` updated with anonymous summaries of feedback

### Phase 5 — Public Archival Release

- [ ] Zenodo DOI assigned for a versioned release of the repository
- [ ] arXiv-ready PDF prepared (correct formatting, metadata, abstract)
- [ ] arXiv submission made; identifier recorded in `CITATION.cff`
- [ ] Recorded lecture or walkthrough (video or detailed notes)

### Phase 6 — Journal Submission

- [ ] Target journal identified (appropriate for long analytical papers in number theory)
- [ ] Manuscript formatted according to journal style
- [ ] Cover letter prepared
- [ ] Submission made; referee outreach documented if appropriate

### Communication Milestones

- [ ] Recorded lecture or walkthrough video published

---

## Notes on Timing

No specific timeline is committed to here. The priority is correctness and
honesty, not speed. Each phase will be considered complete only when its
checklist items are genuinely satisfied, not merely when content has been
produced.
